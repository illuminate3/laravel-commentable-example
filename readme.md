# <a name="getting-started"></a>Getting Started

1. Clone the repository and configure the application.
    ```shell
    $ git clone git@github.com:Slynova-Org/laravel-commentable-example.git
    ```

2. Run migrations.
    ```shell
    $ php artisan migrate
    ```

3. Run seeders.
    ```shell
    $ php artisan db:seed
    ```

4. Enjoy and ask questions!
